## Why This Changed

ManManV2 provides a cleaner, more modern setup with better documentation. The old ManMan V1 services are in maintenance mode.

### What's Different

**V2 Benefits:**
- Go-based services (better performance, easier deployment)
- Split-plane architecture (control plane + execution plane)
- Clear local development flow with Tilt
- gRPC-based API with Protocol Buffers
- Better organized documentation
